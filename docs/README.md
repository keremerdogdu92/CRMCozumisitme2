# Dokümantasyon

Bu klasör, yönetici ve personel kılavuzları ile teknik şemaları barındıracaktır. İlerleyen adımlarda aşağıdaki dosyalar eklenecektir:

- `setup.md` — Supabase ve Netlify kurulum yönergeleri
- `admin-manual.md` — Admin kullanıcıları için kullanım kılavuzu
- `staff-manual.md` — Personel kullanıcıları için kullanım kılavuzu
- `technical-spec.md` — Sistem mimarisi ve veri akışı
- `data-dictionary.json` — Tablo ve alan açıklamaları
